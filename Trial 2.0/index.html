About us

We are Team Bill! We are a project team in Husky Coding Project at the University of Washington. Our team has 7 members and we are all interested in learning about the computer science industry. Our goal for this project was to have fun and get exposed to some new programming concepts! We also love music analyzing tools such as Spotify Wrapped, so we wanted to make our own version for others to enjoy and learn more about their own music tastes.
To create this project we used Firebase as our hosting service, and we used Spotify's Web API to get data about our users.
